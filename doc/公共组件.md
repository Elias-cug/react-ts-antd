### 使用指南

### api 文档

无特别说明的 api 全部可以对照 ant-d 文档使用

1. 新增 api 

| 参数 | 说明 | 类型 | 默认值 | 版本 |
| :-- | :-- | :--| : --| :-- |
| allowCreate | 是否允许创建条目。 | boolean | false |  |

2. 删除 api

| 参数 | 说明                                                         |
| ---- | ------------------------------------------------------------ |
| mode | 此参数通过 allowCreate和属性 showSearch 共同决定，在组件内部判断，无需设置 |

3. 特别说明

| 参数       | 说明                                                         | 默认值 |
| ---------- | ------------------------------------------------------------ | ------ |
| showSearch | 此参数控制是否可输入，allowCerate=true 时，此参数自动设置为 true | false  |
